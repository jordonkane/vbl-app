## Overview :sunflower:
The app allows the user to create and list vocabulary words.

### Technologies Used:
- Spring Boot
- MySQL
- Gradle
- Android Studio IDE

## Usage :wrench:
| Endpoint | Description |
| --- | --- |
| `/news` | Displays esports-related articles from all sites. |
| `/news/{id}` | Displays esports-related articles from a *specific* site.
